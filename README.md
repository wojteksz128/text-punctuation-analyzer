# Text punctuation analyzer

## Requirements

1. Apache Maven >= 3.6
2. Kotlin + JVM 1.3
3. SWI-Prolog
4. JPL >= 7.6.0

## Building

Use Apache Maven to build app, using command:
```shell script
mvn clean install -Djpl.path=<jpl_jar>
```
replacing `<jpl_jar>` with the path to the _jpl-7.6.0.jar_ file.
